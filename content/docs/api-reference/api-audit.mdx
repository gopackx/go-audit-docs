---
title: API Audit API
description: Record and query outbound HTTP calls.
---

## `Auditor.API().Record(ctx, entry) → error`

Writes an API call record to the `audit_api_logs` table. Applies
redaction and truncation according to `APIAuditConfig`.

```go
_ = auditor.API().Record(ctx, audit.APIEntry{
    Service:    "bca",
    Endpoint:   "/v1/transfer",
    Method:     http.MethodPost,
    StatusCode: 200,
    RequestHeaders: map[string]any{
        "Authorization": "Bearer " + token,
        "Content-Type":  "application/json",
    },
    RequestBody:  reqBody,
    ResponseBody: respBody,
    DurationMs:   842,
})
```

## `Auditor.API().Query(ctx, filter) → ([]AuditAPILog, error)`

Returns API call records matching the filter, ordered by
`created_at DESC`.

## `audit.APIEntry`

```go
type APIEntry struct {
    Service        string
    Endpoint       string
    Method         string
    StatusCode     int
    RequestHeaders map[string]any
    RequestBody    any
    ResponseBody   any
    DurationMs     int
    ErrorMessage   string
    Metadata       map[string]any
}
```

## `audit.APIFilter`

```go
type APIFilter struct {
    Service    string
    Endpoint   string
    Method     string
    StatusCode int
    UserID     string
    DateFrom   time.Time
    DateTo     time.Time
    Limit      int
    Offset     int
}
```

## `audit.AuditAPILog`

```go
type AuditAPILog struct {
    ID             uint64         `json:"id"`
    Service        string         `json:"service"`
    Endpoint       string         `json:"endpoint"`
    Method         string         `json:"method"`
    StatusCode     int            `json:"status_code"`
    RequestHeaders map[string]any `json:"request_headers"`
    RequestBody    map[string]any `json:"request_body"`
    ResponseBody   map[string]any `json:"response_body"`
    DurationMs     int            `json:"duration_ms"`
    ErrorMessage   string         `json:"error_message"`
    UserID         string         `json:"user_id"`
    Metadata       map[string]any `json:"metadata"`
    TransactionID  string         `json:"transaction_id"`
    CreatedAt      time.Time      `json:"created_at"`
}
```
